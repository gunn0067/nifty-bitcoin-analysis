# Nifty 50 vs Bitcoin — Indian Market Analysis (2017–2022)

## What is this project?
Ever wondered — if you invested ₹1,000 every month since 2017, 
how much would you have today in Nifty 50 vs Bitcoin?

This project answers that — and goes deeper into market fear, 
sector performance, and COVID crash recovery using real Indian market data.

## Key Findings
- ₹62,000 invested → Nifty gave ₹95,365 (54% profit)
- ₹62,000 invested → Bitcoin gave ₹4,43,239 (615% profit)
- Bitcoin crashed 83% during COVID. Nifty crashed 45%.
- IT sector gave 157% returns from COVID crash bottom. FMCG only 57%.
- When India VIX (Fear Index) spiked to 83.6 — it was actually the best time to invest.
